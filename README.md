# Enhanced Eye Gaze Based Mouse Control System

## Overview

This project enables computer mouse control using eye movements using computer vision and facial landmark detection.

## Technologies Used

* Python
* OpenCV
* MediaPipe

## Features

* Eye tracking
* Cursor movement control
* Blink detection
* Fatigue detection

## Project Structure

* main.py
* eye_tracker.py
* blink_detection.py
* cursor_controller.py
* fatigue_detection.py
* utils.py
